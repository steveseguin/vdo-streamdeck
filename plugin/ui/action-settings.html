PTZ Key

Relative values are signed deltas. Absolute values use value2=abs. Guest exposure and local autofocus are blocked because VDO.Ninja does not expose those command paths today.

PTZ Dial

Dial turns send relative PTZ deltas, accumulated and rate-limited before sending to VDO.Ninja. Guest exposure is blocked because VDO.Ninja does not expose a remote exposure command today.

Value Dial

Volume is percent, panning is 0-180, bitrate is kbps, and buffer delay is milliseconds. Buffer delay uses VDO.Ninja's value2 target path, so this first pass supports default or all-stream local buffer control.
